# Trees Project
This is my solution for the Trees Project in ACS 3310: Advanced Trees & Sorting Algorithms. My solution uses an AVL tree to improve the performance of my tweet generator assignment's markov chain.

## To run
After downloading this repo locally, create and launch a virtual environemnt with and install the neccessary packages with

```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Start the flask app with `python app.py` and navigate to the browser to see the interface.
